## 系統功能說明

這套系統會在每天早上 **6:00（台灣時間）** 自動執行，抓取以下資料並推送到你的 Telegram：

| 資料項目 | 說明 |
|---|---|
| 🌐 TSM ADR | 台積電美股 ADR 收盤價與漲跌幅 |
| 📈 費半 SOXX | 費城半導體指數 ETF |

| 📊 綜合情緒評分 | 0～100 分，自動判斷多空訊號 |

## 第一步：建立 GitHub 帳號與 Repo

### 1-1 註冊 GitHub（已有帳號可跳過）
1. 前往 [https://github.com](https://github.com)
2. 點右上角 **Sign up**，輸入 Email、密碼、使用者名稱完成註冊

### 1-2 建立新 Repo
1. 登入後點右上角 **+** → **New repository**
2. 填寫以下設定：
   - **Repository name**：`sentiment-radar`
   - **Visibility**：選 `Public`（公開，Actions 免費額度較多）
   - 勾選 **Add a README file**
3. 點 **Create repository**

---

## 第二步：上傳專案檔案

Repo 需要包含以下三個檔案：

```
sentiment-radar/
├── radar_script.py          ← 主程式
├── requirements.txt         ← 套件清單
└── .github/
    └── workflows/
        └── radar.yml        ← 自動執行排程
```

### 2-1 上傳 `requirements.txt`
在 Repo 頁面點 **Add file** → **Create new file**，檔名填 `requirements.txt`，內容：

```
yfinance
pandas
requests
```

點 **Commit changes** 儲存。

### 2-2 上傳 `radar_script.py`
同上方式新增檔案，檔名 `radar_script.py`，內容貼上完整程式碼（見附錄一）。

### 2-3 建立 GitHub Actions 排程檔
新增檔案，路徑填：`.github/workflows/radar.yml`，內容：

```yaml
name: Market Sentiment Radar

on:
  schedule:
    - cron: '0 22 * * *'   # 每天 UTC 22:00 = 台灣時間 06:00
  workflow_dispatch:        # 允許手動執行

jobs:
  run-radar:
    runs-on: ubuntu-latest
    permissions:
      contents: write

    steps:
      - name: Checkout code
        uses: actions/checkout@v4

      - name: Set up Python 3.11
        uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Install dependencies
        run: pip install -r requirements.txt

      - name: Run Sentiment Radar
        env:
          TG_BOT_TOKEN: ${{ secrets.TG_BOT_TOKEN }}
          TG_CHAT_ID: ${{ secrets.TG_CHAT_ID }}
        run: python radar_script.py

      - name: Backup data to GitHub Repo
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "github-actions[bot]@users.noreply.github.com"
          git add data/
          git diff --staged --quiet || git commit -m "Auto backup: $(date '+%Y-%m-%d %H:%M')"
          git push
```

## 第三步：建立 Telegram 機器人

### 3-1 找到 BotFather
1. 開啟 Telegram（手機或網頁版 [https://web.telegram.org](https://web.telegram.org)）
2. 搜尋 `@BotFather`，點進去後按 **START**

### 3-2 建立新機器人
1. 輸入 `/newbot`
2. 輸入機器人**顯示名稱**（例如：`情緒雷達每日戰報`）
3. 輸入機器人**帳號名稱**（英文，結尾必須是 `bot`，例如：`SentimentRadarLG_bot`）
4. BotFather 會回覆一串 **Token**，格式如：
   ```
   1234567890:AAGxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   ⚠️ **請完整複製這串 Token，注意英文大小寫（O 和 0 要分清楚）**

### 3-3 取得你的 Chat ID
1. 在 Telegram 搜尋你剛建立的機器人名稱，點進去按 **START**
2. 開啟瀏覽器，前往以下網址（把 `YOUR_TOKEN` 換成你的 Token）：
   ```
   https://api.telegram.org/botYOUR_TOKEN/getUpdates
   ```
3. 在回傳的 JSON 中找 `"chat":{"id":` 後面的數字，那就是你的 **Chat ID**

## 第四步：設定 GitHub Secrets

這步驟是把 Telegram Token 和 Chat ID 安全地儲存在 GitHub，讓 Actions 可以使用。

1. 進入你的 Repo 頁面
2. 點上方選單 **Settings**
3. 左側點 **Secrets and variables** → **Actions**
4. 點 **New repository secret**，新增以下兩個：

| Name | Value |
|---|---|
| `TG_BOT_TOKEN` | 你的 Telegram Bot Token（完整複製，勿漏字）|
| `TG_CHAT_ID` | 你的 Chat ID（數字）|

> ⚠️ **重要**：Token 中若有英文字母 O，要確認不是數字 0。建議用 JavaScript 方法取得（見附錄二）。

## 常見問題排除

### ❌ Telegram 出現 401 Unauthorized
**原因**：Token 輸入錯誤（最常見是 O 和 0 混淆）  
**解法**：
1. 前往 Telegram Web，找 BotFather 的對話
2. 在瀏覽器 Console 執行以下 JavaScript，用程式讀取 Token（避免肉眼誤讀）：
   ```javascript
   var walker = document.createTreeWalker(document.body, NodeFilter.SHOW_TEXT);
   var node; var results = [];
   while(node = walker.nextNode()) {
     if(node.textContent.match(/\d{10}:[A-Za-z0-9_-]{35}/)) {
       results.push(node.textContent.trim());
     }
   }
   console.log(results);
   ```
3. 複製 Console 輸出的 Token，重新設定 GitHub Secret

### ❌ 籌碼動向顯示 N/A
**原因**：台灣證交所資料只在交易日收盤後才有，週末或假日查不到當日資料  
**解法**：程式已內建「往前最多查 5 個日曆日」的重試邏輯，若仍顯示 N/A 表示最近 5 天都是假日或 API 暫時故障，屬正常現象。

### ❌ Actions 執行失敗（紅色 X）
1. 點進去查看 **Run Sentiment Radar** 步驟的 log
2. 確認 Secrets 名稱是否完全一致（`TG_BOT_TOKEN`、`TG_CHAT_ID`）
3. 確認 `requirements.txt` 內容正確

## 附錄一：完整 radar_script.py

```python
import os
import requests
import yfinance as yf
import pandas as pd
from datetime import datetime
import json

# 從環境變數讀取金鑰 (GitHub Secrets)
TELEGRAM_BOT_TOKEN = os.getenv("TG_BOT_TOKEN")
TELEGRAM_CHAT_ID = os.getenv("TG_CHAT_ID")


def send_telegram_msg(message):
    """發送 Telegram 訊息"""
    url = f"https://api.telegram.org/bot{TELEGRAM_BOT_TOKEN}/sendMessage"
    payload = {
        "chat_id": TELEGRAM_CHAT_ID,
        "text": message,
        "parse_mode": "Markdown"
    }
    response = requests.post(url, json=payload)
    if response.status_code == 200:
        print("✅ Telegram 訊息發送成功")
    else:
        print(f"❌ Telegram 發送失敗: {response.text}")


def get_us_market_sentiment():
    """1. 國際半導體情緒：TSM ADR 溢價與 SOX 指數"""
    try:
        tsm = yf.Ticker("TSM")
        soxx = yf.Ticker("SOXX")
        tsm_hist = tsm.history(period="2d")
        tsm_price = round(tsm_hist['Close'].iloc[-1], 2)
        tsm_prev = round(tsm_hist['Close'].iloc[-2], 2) if len(tsm_hist) >= 2 else tsm_price
        tsm_change_pct = round((tsm_price - tsm_prev) / tsm_prev * 100, 2)
        soxx_hist = soxx.history(period="2d")
        soxx_price = round(soxx_hist['Close'].iloc[-1], 2)
        soxx_prev = round(soxx_hist['Close'].iloc[-2], 2) if len(soxx_hist) >= 2 else soxx_price
        soxx_change_pct = round((soxx_price - soxx_prev) / soxx_prev * 100, 2)
        return {
            "TSM_ADR_Price": tsm_price,
            "TSM_ADR_Change": f"{tsm_change_pct:+.2f}%",
            "SOXX_Price": soxx_price,
            "SOXX_Change": f"{soxx_change_pct:+.2f}%"
        }
    except Exception as e:
        print(f"⚠️ 美股資料抓取失敗: {e}")
        return {"TSM_ADR_Price": "N/A", "TSM_ADR_Change": "N/A",
                "SOXX_Price": "N/A", "SOXX_Change": "N/A"}


def get_vix_index():
    """2. 恐慌指數 VIX"""
    try:
        vix = yf.Ticker("^VIX")
        vix_hist = vix.history(period="1d")
        vix_value = round(vix_hist['Close'].iloc[-1], 2)
        if vix_value < 15:
            vix_signal = "😌 低恐慌（偏多頭）"
        elif vix_value < 25:
            vix_signal = "😐 正常波動"
        elif vix_value < 35:
            vix_signal = "😨 恐慌升溫（警戒）"
        else:
            vix_signal = "🚨 極度恐慌（可能反彈）"
        return {"VIX": vix_value, "VIX_Signal": vix_signal}
    except Exception as e:
        print(f"⚠️ VIX 資料抓取失敗: {e}")
        return {"VIX": "N/A", "VIX_Signal": "N/A"}


def get_twse_chips():
    """3. 籌碼面情緒：三大法人（以 2330 台積電為例）"""
    from datetime import timedelta
    headers = {"User-Agent": "Mozilla/5.0"}
    # 往前最多查 5 個日曆日，找到有資料的最近交易日
    for days_back in range(0, 6):
        check_date = datetime.now() - timedelta(days=days_back)
        date_str = check_date.strftime("%Y%m%d")
        try:
            url = f"https://www.twse.com.tw/rwd/zh/fund/T86?response=json&date={date_str}&selectType=ALLBUT0999"
            resp = requests.get(url, headers=headers, timeout=10)
            data = resp.json()
            if not data.get("data"):
                print(f"⚠️ {date_str} 無籌碼資料，往前找...")
                continue
            tsmc_row = None
            for row in data["data"]:
                if "2330" in row[0]:
                    tsmc_row = row
                    break
            if tsmc_row:
                foreign_net = tsmc_row[4].replace(",", "")
                trust_net = tsmc_row[7].replace(",", "")
                dealer_net = tsmc_row[10].replace(",", "")
                total_net = tsmc_row[11].replace(",", "")
                label = check_date.strftime("%m/%d")
                print(f"✅ 找到籌碼資料：{date_str}")
                return {
                    "Target": f"2330 台積電 ({label})",
                    "Foreign_Net": f"{int(foreign_net):+,} 張",
                    "Trust_Net": f"{int(trust_net):+,} 張",
                    "Dealer_Net": f"{int(dealer_net):+,} 張",
                    "Total_Net": f"{int(total_net):+,} 張",
                }
        except Exception as e:
            print(f"⚠️ 籌碼資料抓取失敗 ({date_str}): {e}")
            continue
    return {
        "Target": "2330 台積電", "Foreign_Net": "N/A",
        "Trust_Net": "N/A", "Dealer_Net": "N/A", "Total_Net": "N/A",
    }


def get_etf_premium():
    """4. ETF 溢價率：元大台灣50 (0050)"""
    try:
        url = "https://www.yuantaetfs.com/api/StkRatio"
        headers = {"User-Agent": "Mozilla/5.0"}
        resp = requests.get(url, headers=headers, timeout=10)
        if resp.status_code == 200:
            etf_data = resp.json()
            for item in etf_data:
                if "0050" in str(item):
                    return {"ETF_0050_Premium": str(item)}
        return {"ETF_0050_Premium": "N/A"}
    except Exception as e:
        print(f"⚠️ ETF 資料抓取失敗: {e}")
        return {"ETF_0050_Premium": "N/A"}


def calculate_sentiment_score(us_data, vix_data, chips_data):
    """5. 綜合情緒評分 (0~100 分)"""
    score = 50
    try:
        vix_val = float(vix_data.get("VIX", 20))
        if vix_val < 15:
            score += 15
        elif vix_val < 20:
            score += 5
        elif vix_val > 30:
            score -= 15
        elif vix_val > 25:
            score -= 8
    except:
        pass
    try:
        tsm_chg = float(us_data.get("TSM_ADR_Change", "0%").replace("%", "").replace("+", ""))
        if tsm_chg > 2:
            score += 15
        elif tsm_chg > 0:
            score += 5
        elif tsm_chg < -2:
            score -= 15
        elif tsm_chg < 0:
            score -= 5
    except:
        pass
    score = max(0, min(100, score))
    if score >= 70:
        signal = "🟢 偏多頭，情緒樂觀"
    elif score >= 50:
        signal = "🟡 中性偏多，觀望為主"
    elif score >= 30:
        signal = "🟠 中性偏空，謹慎操作"
    else:
        signal = "🔴 空頭情緒，注意風險"
    return {"Score": score, "Signal": signal}


def run_radar():
    print(f"🚀 啟動多維度情緒雷達 - {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    us_data = get_us_market_sentiment()
    vix_data = get_vix_index()
    chips_data = get_twse_chips()
    etf_data = get_etf_premium()
    sentiment = calculate_sentiment_score(us_data, vix_data, chips_data)
    msg = (
        f"🚀 *多維度情緒雷達戰報*\n"
        f"📅 {datetime.now().strftime('%Y-%m-%d %H:%M')}\n"
        f"{'─'*28}\n\n"
        f"🌐 *美股映射*\n"
        f"• TSM ADR：${us_data['TSM_ADR_Price']}  `{us_data['TSM_ADR_Change']}`\n"
        f"• 費半SOXX：${us_data['SOXX_Price']}  `{us_data['SOXX_Change']}`\n\n"
        f"😨 *恐慌指數 VIX*\n"
        f"• VIX：{vix_data['VIX']}  →  {vix_data['VIX_Signal']}\n\n"
        f"💰 *籌碼動向 ({chips_data['Target']})*\n"
        f"• 外資：{chips_data['Foreign_Net']}\n"
        f"• 投信：{chips_data['Trust_Net']}\n"
        f"• 自營：{chips_data['Dealer_Net']}\n"
        f"• 合計：{chips_data['Total_Net']}\n\n"
        f"📊 *綜合情緒評分*\n"
        f"• 分數：{sentiment['Score']} / 100\n"
        f"• 訊號：{sentiment['Signal']}\n\n"
        f"💾 資料已備份至 GitHub Actions"
    )
    print(msg)
    if TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID:
        send_telegram_msg(msg)
    else:
        print("⚠️ 未設定 Telegram 金鑰，跳過發送")
    result = {
        "timestamp": datetime.now().isoformat(),
        "us_market": us_data,
        "vix": vix_data,
        "chips": chips_data,
        "etf": etf_data,
        "sentiment": sentiment
    }
    output_path = f"data/sentiment_{datetime.now().strftime('%Y%m%d_%H%M')}.json"
    with open(output_path, "w", encoding="utf-8") as f:
        json.dump(result, f, indent=4, ensure_ascii=False)
    print(f"✅ 資料已儲存：{output_path}")


if __name__ == "__main__":
    run_radar()
```

## 附錄二：用 JavaScript 安全讀取 Telegram Token

當懷疑 Token 有 O/0 混淆問題時，在瀏覽器 Console 執行：

```javascript
var walker = document.createTreeWalker(document.body, NodeFilter.SHOW_TEXT);
var node;
while (node = walker.nextNode()) {
  if (node.textContent.match(/\d{8,12}:[A-Za-z0-9_-]{30,}/)) {
    console.log("Token:", node.textContent.trim());
  }
}
```

這個方式直接讀取 DOM 文字節點，不會因為字型導致肉眼混淆。
