# 自動化台股戰情系統 — 完整安裝教程

> **適用對象**：具備基本電腦操作能力，無需程式設計背景。  
> **完成後效果**：每天自動抓取台股數據 → Gemini AI 分析 → Telegram 推播晨間戰鬥清單與夜間覆盤報告。  
> **所需時間**：約 60～90 分鐘（首次設定）。  
> **費用**：完全免費（使用 Google 免費服務與 Gemini 免費額度）。

---

## 系統架構概覽

```
[台灣證交所 API]
       ↓ 每日 14:00 自動抓取
[Google 試算表] ←→ [Google Apps Script (GAS)]
       ↓                      ↓
  儲存股價數據         呼叫 Gemini AI 分析
                             ↓
                    [Telegram Bot 推播]
                      ↙           ↘
               晨間戰鬥清單      夜間覆盤報告
               (07:00 推播)     (21:00 推播)
```

## 步驟一：建立 Google 試算表（系統大腦）

### 1-1 新增試算表

1. 打開電腦，前往 [Google 雲端硬碟](https://drive.google.com)。
2. 點擊左上角 **「+ 新增」** → **「Google 試算表」**。
3. 將試算表標題命名為：`自動化台股戰情室`。

### 1-2 建立四個工作表分頁

在試算表左下角，連續點擊 **「+」** 號新增分頁，建立以下四個分頁（名稱必須完全一致）：

| 分頁名稱 | 用途 |
|----------|------|
| `追蹤清單` | 記錄你想關注的股票與理由 |
| `每日行情` | 自動寫入每日收盤數據 |
| `國際盤勢` | 放置美股即時公式 |
| `戰情總覽` | 整合查詢視圖與趨勢圖 |

> ⚠️ **注意**：分頁名稱不能有多餘的空格或錯字，程式碼會依此名稱尋找分頁。

### 1-3 填入「每日行情」欄位標題

點開 **「每日行情」** 分頁，在第一列輸入標題：

| A1 | B1 | C1 | D1 | E1 | F1 |
|----|----|----|----|----|-----|
| 日期 | 代號 | 名稱 | 收盤價 | 漲跌價 | 成交股數 |

### 1-4 設定「國際盤勢」即時公式

點開 **「國際盤勢」** 分頁，依序輸入以下內容：

| 欄位 | 內容 |
|------|------|
| A1 | `指標` |
| B1 | `最新價格` |
| A2 | `台積電 ADR` |
| A3 | `標普 500` |
| A4 | `納斯達克` |
| B2 | `=GOOGLEFINANCE("NYSE:TSM", "price")` |
| B3 | `=GOOGLEFINANCE(".INX", "price")` |
| B4 | `=GOOGLEFINANCE("INDEXNASDAQ:.IXIC", "price")` |

輸入完成後，B2～B4 會自動顯示即時美股數字（可能需要幾秒載入）。

## 步驟二：申請 Telegram 機器人（通訊介面）

你需要取得兩把「鑰匙」：**Bot Token**（機器人身分證）和 **Chat ID**（你的收件地址）。

### 2-1 建立機器人 & 取得 Token

1. 打開 **Telegram**，在搜尋欄輸入 `@BotFather`（官方帳號，有藍色打勾圖示）。
2. 點擊進入對話，點擊 **「Start」**。
3. 輸入 `/newbot` 並送出。
4. BotFather 會詢問機器人的**顯示名稱**，輸入任意名稱（例如：`我的台股小幫手`）送出。
5. 接著輸入機器人的**帳號 ID**，規則如下：
   - 只能用英文與數字。
   - **結尾必須是 `bot`**（例如：`MyTWStock_bot`）。
   - 如果系統回應 `Sorry, this username is already taken.` 代表已被使用，請換一個名稱再試。
6. 成功後，BotFather 會回傳一大段文字，找到這行：  
   `Use this token to access the HTTP API:`  
   下方那串亂碼就是你的 **Token**（格式範例：`123456789:ABCdefGHIjklMNO...`）。

> **立刻將 Token 複製貼到記事本備用！**

### 2-2 取得你的 Chat ID

1. 在 Telegram 搜尋欄搜尋 `@userinfobot`。
2. 點擊進入對話並按 **「Start」**。
3. 它會回傳你的個人資訊，其中 `Id:` 後面的數字就是你的 **Chat ID**（例如：`987654321`）。

> **將 Chat ID 也複製貼到記事本備用！**

### 2-3 喚醒機器人（重要）

1. 在 Telegram 搜尋欄搜尋你剛建立的機器人帳號（例如：`@MyTWStock_bot`）。
2. 點擊進入對話，按下 **「Start」**。

> ⚠️ 這個步驟**不能略過**，沒有先按 Start 的話，機器人將無法主動傳訊息給你。

## 步驟三：申請 Gemini AI 金鑰（分析核心）

### 3-1 建立 Gemini API Key

1. 前往 [Google AI Studio](https://aistudio.google.com/)。
2. 登入你的 Google 帳號。
3. 點擊頁面上的 **「Create API key」**。
4. 點擊 **「Create API key in new project」**。
5. 畫面跳出一串英數亂碼，這就是你的 **Gemini API Key**。

> **將 API Key 複製貼到記事本備用！**

## 步驟四：進入 Apps Script 後台（寫入程式碼）

### 4-1 開啟 GAS 後台

1. 回到你的「自動化台股戰情室」Google 試算表。
2. 點擊頂部選單 **「擴充功能」** → **「Apps Script」**。
3. 瀏覽器會開一個新分頁，這就是 **Google Apps Script (GAS)** 的程式後台。

### 4-2 貼上完整程式碼

1. 在 GAS 編輯器中，點擊程式碼區域，按 **`Ctrl + A`** 全選，再按 **`Delete`** 清空。
2. 將以下**完整程式碼**複製並貼上：

```javascript
// ==========================================
// ⚙️ 齒輪一：每日自動抓取收盤資料
// ==========================================
function fetchStockData() {
  var url = "https://openapi.twse.com.tw/v1/exchangeReport/STOCK_DAY_ALL";
  var response = UrlFetchApp.fetch(url);
  var data = JSON.parse(response.getContentText());
  
  // 追蹤的股票清單（台積電、台達電、鴻海、聯發科、日月光、廣達、台光電、富邦金、國泰金、中信金）
  var myStocks = ["2330", "2308", "2317", "2454", "3711", "2382", "2881", "2383", "2882", "2891"];
  
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("每日行情");
  var today = Utilities.formatDate(new Date(), "GMT+8", "yyyy-MM-dd");

  data.forEach(function(stock) {
    if (myStocks.indexOf(stock.Code) !== -1) {
      sheet.appendRow([
        today, 
        stock.Code, 
        stock.Name, 
        stock.ClosingPrice, 
        stock.Change, 
        stock.TradeVolume
      ]);
    }
  });
}

function nightlyReview() {
  // ⚠️ 請填入你的三把金鑰
  var telegram_token = "請換成你的 Bot Token";
  var telegram_chatId = "請換成你的 Chat ID";
  var gemini_apiKey = "請換成你的 Gemini API Key";

  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("每日行情");
  var lastRow = sheet.getLastRow();
  var data = sheet.getRange(lastRow, 1, 1, 6).getValues()[0];
  
  var date = data[0];
  var stockName = data[2];
  var price = data[3];
  var change = data[4];

  var prompt = `你是專業的台股分析師。請根據以下當日數據，寫一份 150 字內的簡短夜間覆盤報告：
  - 日期：${date}
  - 股票：${stockName}
  - 收盤價：${price}
  - 漲跌：${change}
  
  報告請包含：1.盤勢簡評 2.隔日操作建議。
  ⚠️排版嚴格規定：回覆內容請一律使用純文字，不要使用任何 Markdown 格式符號。`;

  var geminiUrl = "https://generativelanguage.googleapis.com/v1beta/models/gemini-flash-latest:generateContent?key=" + gemini_apiKey;
  var payload = { "contents": [{ "parts": [{ "text": prompt }] }] };
  
  var response = UrlFetchApp.fetch(geminiUrl, {
    "method": "post",
    "contentType": "application/json",
    "payload": JSON.stringify(payload),
    "muteHttpExceptions": true
  });
  
  if (response.getResponseCode() !== 200) {
    Logger.log("Gemini API 呼叫失敗：" + response.getContentText());
    return;
  }

  var aiResult = JSON.parse(response.getContentText());
  var aiReport = aiResult.candidates[0].content.parts[0].text;

  var message = "📊 【夜間覆盤報告】\n\n" + aiReport;
  var telegramUrl = "https://api.telegram.org/bot" + telegram_token + "/sendMessage";
  UrlFetchApp.fetch(telegramUrl, {
    "method": "post",
    "contentType": "application/json",
    "payload": JSON.stringify({ "chat_id": telegram_chatId, "text": message })
  });
}
```

## 步驟七：設定「戰情總覽」看板（視覺化）

### 7-1 QUERY 即時排行榜

點開「戰情總覽」分頁，在 **A1** 貼上以下公式：

```
=QUERY('每日行情'!A:F, "SELECT B, C, D, E, F ORDER BY A DESC LIMIT 10", 1)
```

這會自動顯示最新的 10 檔股票數據，並從新到舊排序。

### 7-2 Sparkline 迷你走勢圖

在「戰情總覽」公式結果的旁邊欄（例如 F 欄），在 **F2** 輸入：

```
=SPARKLINE(QUERY('每日行情'!A:F, "SELECT D WHERE B = '"&B2&"' ORDER BY A DESC LIMIT 5", 0))
```

再將 F2 往下拖曳到 F11，每行股票旁邊就會出現最近 5 天的迷你走勢線。

### 7-3 漲跌顏色條件式格式（選用）

1. 選取「戰情總覽」的漲跌欄（E2:E11）。
2. 點擊選單 **「格式」** → **「條件式格式」**。
3. 新增規則：「大於 0」→ 背景色設為**紅色**。
4. 再新增規則：「小於 0」→ 背景色設為**綠色**。

## 常見問題 FAQ

### Q1：執行時出現 404 Not Found（Gemini API）

**原因**：模型名稱可能已更新。  
**解決**：確認程式碼中使用的是 `gemini-flash-latest`。  
若仍失敗，在 GAS 中新增以下函數查詢可用模型清單：

```javascript
function checkAvailableModels() {
  var gemini_apiKey = "你的 API Key";
  var url = "https://generativelanguage.googleapis.com/v1beta/models?key=" + gemini_apiKey;
  var response = UrlFetchApp.fetch(url, { "method": "get", "muteHttpExceptions": true });
```

執行後，在執行記錄中找到 `"name": "models/gemini-..."` 的名稱，替換程式碼中的模型名稱。

### Q2：Telegram 收到錯誤 `can't parse entities`

**原因**：AI 回覆使用了 Markdown 符號（星號 `*`），但 Telegram 找不到配對的結尾符號。  
**解決**：確認發送 Telegram 訊息的程式碼中**沒有** `"parse_mode": "Markdown"` 這行，改為純文字發送即可。

### Q3：試算表「每日行情」沒有資料

**原因**：台灣證交所 API 只在收盤後（下午 1:30 後）才有當日數據。  
**測試方法**：在非交易時段可手動在「每日行情」輸入測試資料，先驗證公式是否正常運作。  
或使用 GoogleFinance 公式驗證：

```
D2 = =GOOGLEFINANCE("TPE:2330", "price")
```
